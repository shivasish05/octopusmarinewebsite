# Octopus Supply Chain Website

A modern, responsive website for Octopus Supply Chain logistics company.

## Overview

This website showcases Octopus Supply Chain's logistics and supply chain management services, featuring:
- Modern, responsive design
- Interactive UI elements
- Partner showcase
- Service descriptions
- Contact forms
- Live chat widget
- Interactive maps

## Structure
```
├── assets/
│   ├── octopus-logo.jpg    # Main company logo
│   └── partners/           # Partner logos directory
├── index.html             # Main website file
└── README.md             # This file
```

## Setup Instructions

1. Clone the repository
```bash
git clone <your-repository-url>
cd octopusscmapp
```

2. Add required assets:
- Place company logo as `assets/octopus-logo.jpg`
- Add partner logos in `assets/partners/` directory

3. Open `index.html` in a web browser or serve using a local server
